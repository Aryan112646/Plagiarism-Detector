# 📄  Plagiarism Detector

A full-stack web application that quantifies similarity between text documents using Natural Language Processing (NLP).

## 🚀 Features
* **TF-IDF Analysis:** Uses Term Frequency-Inverse Document Frequency to weigh meaningful words.
* **Cosine Similarity:** Mathematical comparison of document vectors to detect overlap.
* **Scan History:** SQLite database integration to persist and track previous comparisons.
* **PDF Reports:** Generate and download professional analysis reports instantly.
* **Responsive UI:** Modern, clean interface built with CSS3 and Vanilla JavaScript.

## 🛠️ Tech Stack
* **Backend:** Python, Flask, Scikit-learn
* **Frontend:** JavaScript (ES6+), HTML5, CSS3
* **Database:** SQLite3
* **Libraries:** jsPDF, Flask-CORS

## 🔧 Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/Plagiarism-Detector.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
